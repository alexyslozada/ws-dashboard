# ws-dashboard
Dashboard en tiempo real con websockets (js + go)

## Install
```bash
$ go get -u github.com/alexyslozada/ws-dashboard
```

## Build & Execute
```bash
$ cd $GOPATH/src/github.com/alexyslozada/ws-dashboard
$ go build && ./ws-dashboard
```

## Only Build
```bash
$ cd $GOPATH/src/github.com/alexyslozada/ws-dashboard
$ go build
```

## Only Execute
```bash
$ cd $GOPATH/src/github.com/alexyslozada/ws-dashboard
$ ./ws-dashboard
```
